简介
====

CIF全称为Crystallographic Information File，是一种用于表示
晶体信息的标准文本文件格式。该格式由国际晶体学联合会（IUCr）
开发，许多用于分子显示的计算机程序都与此格式兼容，例如
Jmol，VESTA，Materials Studio等。

Materials Project是用于下载晶体CIF的常用数据库。

CIF Tool可以通过筛选搜索获取Materials Project中的结构数据，
并可一键保存搜索得到的结构对应的CIF。

使用方法
========

1. 在Materials Project官网注册账号  
2. 在软件API Key处填写官网获取的API Key  
3. 填写搜索条件，单击Load Data按钮  
4. 待加载完成后，单击Save .cif按钮，即可完成cif文件批量下载
