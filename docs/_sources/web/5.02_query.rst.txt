========
数据查询
========


带隙查询
========

网址：

包含6354个无机晶体结构的带隙数据。

* 化学式：材料的化学式
* 带隙：实验带隙数据，单位eV

生成焓查询
==========

网址：

包含1276个合金的生成焓数据。

* 化学式：合金的化学式，原子数可能包含小数
* 生成焓：单位eV/atom
* 空间群/pearson符号：对应结构的对称性信息
* MPID/MP生成焓：若在MP数据库中存在对应结构，对应结构的ID和生成焓，如不存在，则这两项为nan
* OQMDID/OQMD生成焓：若在OQMD数据库中存在对应结构，对应结构的ID和生成焓，如不存在，则这两项为nan

弹性模量查询
============

网址：

包含1181个DFT计算的无机晶体结构的剪切模量、体积模量、泊松比等数据。

* 化学式：材料的化学式
* 弹性各向异性/泊松比：弹性各向异性与泊松比数据
* 剪切模量下界/平均值/上界：多晶材料剪切模量的下界、平均值、上界，单位GPa
* 体积模量下界/平均值/上界：多晶材料体积模量的下界、平均值、上界，单位GPa

介电常数查询
============

网址：

包含1296个无机晶体结构的介电常数数据以及电子贡献部分。

* 化学式：材料的化学式
* 介电张量：总的介电张量数据，包含电子贡献部分和离子贡献部分
* 电子贡献部分：电子贡献部分的介电张量数据

杂质扩散活化能查询
==================

网址：

包含金属晶体中的加入杂质原子后杂质原子的扩散活化能数据。

* 主体金属：包含Fe,Co,Ni,Cu,Ag,Au六种FCC金属晶体
* 杂质原子：掺杂的杂质原子
* 活化能：杂质原子扩散活化能，单位为kJ/mol
