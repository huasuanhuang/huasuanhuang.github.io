使用方法
========

创建结构
--------
可根据晶体名称，
通过预设数据集里导入晶体数据，
也可直接根据堆积方式与晶胞参数，
创建晶体。

导入结构
--------
可直接导入晶体结构文件数据，
支持格式有.cif，POSCAR，OUTCAR，.xsd，.xyz，.cell。

调整结构
--------
可对创建或导入的晶体结构进行超胞，
固定原子，随机取代的调整。

结构输出
--------
可调用jmol可视化程序中的结构，
也可将结构保存为想要的格式，
支持格式有POSCAR，.cif，.dat，.xyz。

批量转换
--------
可实现批量文件格式转换，
支持的输入格式有.cif，POSCAR，.xsd，
支持的输出格式有POSCAR，.cif，.xyz。
